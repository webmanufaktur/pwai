# wireMethodExists()

**ProcessWire Helper Function**

- [Official Docs](https://processwire.com/api/ref/wiremethodexists/)

## Overview

`wireMethodExists()` checks if a method exists in a class or object.

## Usage

```php
if(wireMethodExists($object, 'myMethod')) {
    // ...
}
```
