# wireEmpty()

**ProcessWire Common Function**

- [Official Docs](https://processwire.com/api/ref/wireempty/)

## Overview

`wireEmpty()` checks if a variable is empty, similar to PHP's `empty()`, but with ProcessWire-specific enhancements.

## Usage

```php
if(wireEmpty($value)) {
    // ...
}
```
