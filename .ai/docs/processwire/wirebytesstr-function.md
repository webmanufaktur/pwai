# wireBytesStr()

**ProcessWire Strings Function**

- [Official Docs](https://processwire.com/api/ref/wirebytesstr/)

## Overview

`wireBytesStr()` formats a byte count into a human-readable string (KB, MB, GB, etc.).

## Usage

```php
echo wireBytesStr(1024); // "1 KB"
```
