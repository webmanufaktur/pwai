# wireClassParents()

**ProcessWire Helper Function**

- [Official Docs](https://processwire.com/api/ref/wireclassparents/)

## Overview

`wireClassParents()` returns an array of parent classes for a given class.

## Usage

```php
$parents = wireClassParents('MyClass');
```
