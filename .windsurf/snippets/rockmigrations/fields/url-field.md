---
title: URL Field
description: Create a URL input field with RockMigrations
tags: [field, url, link, rockmigrations]
---

```php
'website' => [
  'type' => 'url',
  'label' => 'Website',
],
```

**Options:**
- `required`: Whether the field is required (true/false)
- `description`: Help text shown below the field
