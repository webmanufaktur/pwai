---
title: Email Field
description: Create an email input field with RockMigrations
tags: [field, email, rockmigrations]
---

```php
'email' => [
  'type' => 'email',
  'label' => 'Email',
  'required' => true,
],
```

**Options:**
- `required`: Whether the field is required (true/false)
- `description`: Help text shown below the field
