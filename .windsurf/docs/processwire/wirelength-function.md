# wireLength()

**ProcessWire Common Function**

- [Official Docs](https://processwire.com/api/ref/wirelength/)

## Overview

`wireLength()` is an alias of `wireLen()`, returning the length of a string, array, or countable object.

## Usage

```php
$length = wireLength($value);
```
