# wireCopy()

**ProcessWire Files Function**

- [Official Docs](https://processwire.com/api/ref/wirecopy/)

## Overview

`wireCopy()` copies a file or directory with ProcessWire-specific enhancements.

## Usage

```php
wireCopy('/path/to/source', '/path/to/destination');
```
