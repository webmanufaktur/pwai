# wireClassNamespace()

**ProcessWire Helper Function**

- [Official Docs](https://processwire.com/api/ref/wireclassnamespace/)

## Overview

`wireClassNamespace()` returns the namespace of a class.

## Usage

```php
$namespace = wireClassNamespace('MyClass');
```
