# wireSendFile()

**ProcessWire Files Function**

- [Official Docs](https://processwire.com/api/ref/wiresendfile/)

## Overview

`wireSendFile()` sends a file to the browser for download with proper headers.

## Usage

```php
wireSendFile('/path/to/file.pdf');
```
