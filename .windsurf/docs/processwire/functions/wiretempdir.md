# wireTempDir()

**ProcessWire Files Function**

- [Official Docs](https://processwire.com/api/ref/wiretempdir/)

## Overview

`wireTempDir()` returns the path to a temporary directory, creating it if needed.

## Usage

```php
$tempDir = wireTempDir('my-temp-files');
```
