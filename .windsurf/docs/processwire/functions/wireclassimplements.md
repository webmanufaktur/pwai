# wireClassImplements()

**ProcessWire Helper Function**

- [Official Docs](https://processwire.com/api/ref/wireclassimplements/)

## Overview

`wireClassImplements()` checks if a class implements a given interface.

## Usage

```php
if(wireClassImplements('MyClass', 'MyInterface')) {
    // ...
}
```
