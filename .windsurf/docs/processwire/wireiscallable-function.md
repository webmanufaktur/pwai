# wireIsCallable()

**ProcessWire Helper Function**

- [Official Docs](https://processwire.com/api/ref/wireiscallable/)

## Overview

`wireIsCallable()` checks if a function, method, or closure is callable.

## Usage

```php
if(wireIsCallable($callback)) {
    // ...
}
```
