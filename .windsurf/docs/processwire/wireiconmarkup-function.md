# wireIconMarkup()

**ProcessWire Markup Function**

- [Official Docs](https://processwire.com/api/ref/wireiconmarkup/)

## Overview

`wireIconMarkup()` generates HTML markup for an icon based on a name or class.

## Usage

```php
echo wireIconMarkup('file-o');
```
