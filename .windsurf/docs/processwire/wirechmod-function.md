# wireChmod()

**ProcessWire Files Function**

- [Official Docs](https://processwire.com/api/ref/wirechmod/)

## Overview

`wireChmod()` changes file permissions (chmod) with ProcessWire-specific enhancements.

## Usage

```php
wireChmod('/path/to/file', 0644);
```
