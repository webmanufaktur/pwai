# setting()

**ProcessWire Functions-API Function**

- [Official Docs](https://processwire.com/api/ref/setting/)

## Overview

`setting()` is used to get or set configuration settings in ProcessWire.

## Usage

```php
$debugMode = setting('debug');
setting('debug', true);
```
