# wireDate()

**ProcessWire Common Function**

- [Official Docs](https://processwire.com/api/ref/wiredate/)

## Overview

`wireDate()` formats a timestamp or date string according to ProcessWire or PHP date formats.

## Usage

```php
echo wireDate('Y-m-d', time());
```
